Improving poultry health and welfare

As this is a competitive grant, we'll need to check how closely your project matches our funding priorities to improve poultry health and welfare.

What you'll need

You'll need details of how you plan to do the following in your project:

  • improve poultry health and welfare - for example, features to help reduce keel bone fractures
  • reduce pollution and increase biosecurity - for example, adding shower facilities or air outlets
  • use renewable energy sources
  • use digital systems to capture data automatically
How we assess applications

We assess applications based on:

  • evidence your project will improve poultry health and welfare
  • value for money
  • how many applications we receive
